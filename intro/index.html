# Вступ

Мета роботи: розробити систему управління проектами, яка б була краща, ніж існуючі засоби рішення, спираючись на порівняльні дані, які ми зберемо, мала б всі інструменти для комфортної роботи та давала можливість легко реалізувати та налагодити спільну роботу над проектом.

Наразі існує багато різних систем управління проектами з різною функціональністю, проте багато з них мають недостатньо повну функціональність для роботи, застарілі та не конкурентоспроможні. Наш проект намагатиметься задовольнити ключові сучасні потреби користувачів.

На кінець виконання лабораторних робіт основними технічними характеристиками розроблення будуть: модель бізнес-об'єктів, модель прецедентів, ER-модель, реляційна схема для бази даних, SQL script для початкового наповнення бази та Restful service для управління даними.

Оскільки артефакти не дуже розповсюджені, то очікуваний технічно-економічний ефект нашого проекту буде таким, що наша система управління проектами зможе підтримувати їх розробку.
